# A Soft Retake on Heading Home Mode for Rapido

## Overview
This isn’t a full case study, just a quick teardown.

I came across a cool idea while scrolling LinkedIn: Rapido’s “Heading Home” mode, the idea where drivers enter their home or destination before ending a shift, and the app prioritizes ride requests along that route. I wanted to stress‑test the concept using the CIRCLES framework before talking on the build.

The aim was to figure out:

* Who actually faces this problem
* What the root cause is
* What trade‑offs different solutions carry
* What guardrails would be needed before shipping
